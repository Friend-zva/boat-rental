# Boat Rental (24)

## Описание

Система предназначена для автоматизации проката плавательных средств, обеспечивающего отдых граждан в парке с развитой системой живописных водоемов. Плавательные средства – это двух- и четырех- местные лодки с веслами, а также водные велосипеды одного типа, но от разных производителей. Цена (почасовая) проката для лодок и велосипедов различается. Разработанная схема данных должна позволять хранить реестр плавательных средств, включая цвет и степень изношенности, историю проката за последние три месяца (включая количество людей, воспользовавшихся услугами проката), а также минимальную информацию о сотрудниках проката.

## Серверная часть

### Функции

| Функция | Комментарий |
| ------- | ----------- |
| AddEmployee | Регистрация нового работника. |
| AddLessee | Регистрация нового клиента. |
| AddBoat | Регистрация новой лодки. |
| AddWaterBike | Регистрация нового водного велосипеда. |
| UpdDelUnit | Обновление или удаление прокатной единицы. У любой единицы можно изменить цену и процент износа и описать причину. |
| GetFreeUnits | Получить свободные на данный момент прокатные единицы. |

### Триггеры

| Триггер | Комментарий |
| ------- | ----------- |
| TR_Lessee_DEL | Запрет на удаление клиента. |
| TR_Rental_INS | Удаление сведений о прокате старше 3 месяцев после вставки нового элемента. |

### Представления

| Представление | Комментарий |
| ------------- | ----------- |
| V_Rental_Details | Получить всю информацию о заказе. |
| V_Free_Units | Получить свободные на данный момент прокатные единицы. Аналог GetFreeUnits. |
| V_Employee_Stats | Получить статистику работников. |
